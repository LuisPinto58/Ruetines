# Ruetines

<!-- O teu ritmo, sem culpas -->

A *Ruetines* é uma aplicação para ajudar a gerir as rotinas diárias de forma gentil. Foi pensada para pessoas que estão a passar por depressão ou burnout. 



# Funcionalidades Principais

* *Foco no Presente*: Mostra apenas as tarefas do próprio dia para não causar ansiedade ou sobrecarga com listas grandes.

* *Rotinas Prontas e Sugeridas*: Rotinas simples já feitas, para começares sem teres de pensar muito

* *Streaks Seguros*: Ganha medalhas (badges) pela tua consistência. Se precisares de parar um dia para descansar, o teu progresso (streak) não volta ao zero.

* *Comunidade de Apoio*: Um espaço seguro onde podes partilhar o teu progresso e encontrar apoio ao lado de pessoas com experiências semelhantes.



# Acessibilidade

* *Modo de Leitura Inclusiva*: Incluímos um botão para ativar a fonte OpenDyslexic, para facilitar a leitura a utilizadores com dislexia.

* *Visual Calmo*: A interface utiliza apenas cores gentis e calmas. Eliminámos as cores de alerta e os avisos a vermelho para não gerar stress. 



# Tecnologias Utilizadas 

O projeto foi construído usando as seguintes tecnologias:

* *Front-end*: HTML5, CSS3, JavaScript

* *Back-end/Dependências*: Json Server, Json Server Auth, Socket.io, Express

* *Design & UI/UX*: Figma, Bootstrap

# Como testar o projeto

1. Faz clone deste repositório:
    git clone https://github.com/LuisPinto58/Ruetines.git

2. Faça npm install para instalar as dependências necessárias

3. Npm start para rodar o servidor e sockets

4. Rode o index.html com uma extensão como o live server (recomendado: Desative atualização do liveserver na atualização de ficheiros .json ou novas tarefas e mensagens irão fazer refresh da pagina)
